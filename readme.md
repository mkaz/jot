
## Jot.py

A simple command-line tool for taking notes stored in plain text files.


### Setup

* Copy jot.py to $HOME/bin
* Requires python 

### Usage

1. Jot as argument

```
$ jot.py "Here is my note"
```

2. Free form jot
```
$ jot.py
<< Enter Text to Jot (ctrl-d to end) >>
Here is my 
multi-line note.
```
